# ❤️ Heart Disease Prediction Web App

This project predicts the likelihood of heart disease using Machine Learning and a Flask web application.

## 🚀 Features
- User inputs medical parameters
- Machine learning model predicts heart disease risk
- Interactive web interface
- Model comparison using multiple algorithms

## 🧠 Machine Learning Models Used
- Logistic Regression
- Decision Tree
- Naive Bayes
- Random Forest (Final Model)

## 📊 Dataset
UCI Heart Disease Dataset

## 🛠 Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Bootstrap
- Jupyter Notebook
